# Model Manager Project

Example of a complete project using the agent-project-tracker system.

## What's This

A CLI tool for managing OpenClaw models with:
- Add/remove custom models
- Test connections
- Backup/restore config
- TUI menu (raspi-config style)

## Files

- `PROJECT-model-manager.md` - Project tracking
- `model-manager.sh` - The tool

## Usage

```bash
./model-manager.sh menu    # TUI mode
./model-manager.sh list    # CLI mode
```
